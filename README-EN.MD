# 🏆 GitHub Badges and Achievements Guide

This guide explains what all the achievements and highlights you can display on your GitHub profile mean and how you can earn them.

GitHub achievements are digital badges awarded to celebrate your contributions to open source projects and your engagement on the platform. You can hide your badges or make them public from your profile settings.

---

## 🏅 Achievements and How to Earn Them

You can earn these badges through your daily activities on GitHub (opening Pull Requests, writing answers, etc.).

### 🌟 Starstruck
* **How to Earn:** A repository you created must receive a certain number of stars.
* **Tiers:**
  * **Default:** 16 Stars
  * **Bronze:** 128 Stars
  * **Silver:** 512 Stars
  * **Gold:** 4096 Stars

### ⚡ Quickdraw
* **How to Earn:** You earn this if you close an Issue or a Pull Request (PR) **within 5 minutes** of opening it. It has only 1 tier.

### 🤝 Pair Extraordinaire
* **How to Earn:** You need to be tagged as a "co-author" in a successfully merged Pull Request. (Done by adding `Co-authored-by:` to commit messages).
* **Tiers:**
  * **Default:** 1 PR
  * **Bronze:** 10 PRs
  * **Silver:** 24 PRs
  * **Gold:** 48 PRs

### 🦈 Pull Shark
* **How to Earn:** The Pull Requests you open must be successfully merged into the main project.
* **Tiers:**
  * **Default:** 2 PRs
  * **Bronze:** 16 PRs
  * **Silver:** 128 PRs
  * **Gold:** 1024 PRs

### 🧠 Galaxy Brain
* **How to Earn:** Your answers in GitHub Discussions must be marked as the "Accepted Answer" by the person who asked the question.
* **Tiers:**
  * **Default:** 2 Accepted Answers
  * **Bronze:** 8 Accepted Answers
  * **Silver:** 16 Accepted Answers
  * **Gold:** 32 Accepted Answers

### 🚀 YOLO
* **How to Earn:** You earn this badge if you merge a Pull Request you opened directly without receiving any code review from someone else. It has only 1 tier.

### 💖 Public Sponsor
* **How to Earn:** Earned when you sponsor (financially support) an open-source developer or project via the GitHub Sponsors program.

*(Note: Some badges are being tested by GitHub and may not always be earnable).*

---

## ✨ Highlights Badges

These badges represent special statuses and participation on the GitHub platform and are listed in the bottom left corner of your profile.

* **Pro:** You get this badge if you are a GitHub Pro user or have the GitHub Student Developer Pack.
* **Developer Program Member:** Awarded if you are a registered member of the GitHub Developer Program.
* **Security Bug Bounty Hunter:** Awarded to those who contribute to the "Bug Bounty" program by finding security vulnerabilities in GitHub's own systems.
* **GitHub Campus Expert:** Awarded to "GitHub Campus Program" participants who lead student communities.
* **Security Advisory Credit:** Earned when you submit a security advisory that is accepted into the GitHub Advisory Database.
* **GitHub Star:** A highly prestigious badge given to individuals specially selected by GitHub who greatly contribute to and inspire the GitHub community.

These badges were awarded for specific past events and new ones cannot be earned currently:

* **Arctic Code Vault Contributor:** Awarded to those who contributed code to active repositories during the 2020 GitHub Archive Program (The code is stored on film reels in a cave in the Arctic).
* **Mars 2020 Contributor:** Awarded to those who contributed code to open-source repositories of the software used in NASA's Mars 2020 Helicopter (Ingenuity) mission.
